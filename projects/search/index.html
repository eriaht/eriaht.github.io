(Linear Search vs. Binary Search) Visualizer

Linear Search

Linear search will check every element in
a list until the desired element is found.
The runtime of a linear search is BigO(n).
So, if a list is 100 items long, then the
number of search operations is 100.

Binary Search

Binary search starts in the middle
and continues to half the list until
the desired element is found.
The runtime of a binary search is BigO(log2(n)).
So, if a list is 100 items long, then the
number of search operations is only 7.
